## ‚ö° Quick Start (For Cashier)

### Starting the System (Every Day)

1. **Turn on the laptop** (the one connected to the printer)
2. **Turn on the printer** (XP-58 thermal printer - check USB cable connected)
3. **Double-click:** `Start Restaurant POS` (on Desktop)
4. **Wait 10 seconds** - Two windows will open automatically
5. **Done!** The system is ready

### Using the System on iPad

1. **Connect iPad to the same WiFi** as the laptop
2. **Open Safari** on iPad
3. **Go to the website address shown on laptop screen**
   - It will look like: `http://192.168.1.XXX:3000`
4. **Use the POS system normally**
5. **When you print**, receipt will come out of the thermal printer!

### Stopping the System (End of Day)

1. **Double-click:** `STOP-RESTAURANT-POS.bat`
2. **Done!** Everything stops

## ‚ùì Troubleshooting

### Problem: iPad can't open the website

**Solution:**
- Make sure iPad and laptop are on **same WiFi**
- Check the website address is correct
- Try restarting the system (close and start again)

### Problem: Receipt doesn't print

**Solution:**
- Check printer is **ON** and **connected via USB**
- Check printer has **paper**
- Restart the print server (close and start again)

### Problem: Website won't start

**Solution:**
- Close all windows
- Run `STOP-RESTAURANT-POS.bat` first
- Wait 5 seconds
- Then start again

## 🔧 **IF PRINT DOESN'T WORK**

### **Try Method 1: Direct Bluetooth**
1. iPad Settings → Bluetooth
2. Look for "XP-58IIH" or similar
3. Connect to it
4. Return to app → Try print again

### **Try Method 2: Safari Print (Backup)**
1. Click "📄 Safari Print (Backup)"
2. Use iPad's built-in print system
3. Select any available printer

### **Try Method 3: Print Server (Advanced)**
1. Change print method to "🖥️ Laptop Server"
2. Click print again
3. Check if laptop print server is running

### **Troubleshooting:**
- **No Bluetooth connection**: Try iPad Settings → Bluetooth → Connect manually
- **Print fails**: Use "Safari Print (Backup)" button

- **App issues**: Refresh Safari page and try again
